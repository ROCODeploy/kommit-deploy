Zero Data App Ser dono dos seus dados, todos eles.

Princípios

  • um aplicativo no qual os seus dados ficam com você
  • você controla onde os dados são armazenados
  • sem spam, sem captcha, sem conta, sem senha, traga sua própria identidade
  • usando protocolos abertos para flexibilidade e interoperabilidade
  • faça o que você quer com seus dados em qualquer momento
  • seus dados são acessíveis para sempre mesmo qua o aplicativo deixe de funcionar
